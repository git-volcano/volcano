## 🔧 Common Issues and Fixes

### ❌ "Injection Failed" or "Failed to Attach"
**Cause:** Roblox updated after your current Volcano build, or antivirus is blocking the loader process.
**Fix:** Restart Volcano — the auto-updater pulls the latest patch on launch. If still failing, add the Volcano folder to your antivirus exclusion list and relaunch as Administrator.

---

### ❌ Script Executes but Nothing Happens In-Game
**Cause:** The script targets a game feature that has been patched, or it was written for an older Roblox API version.
**Fix:** Test a simple Script Hub entry (speed, jump, or noclip) to confirm the executor is working. If Hub scripts run correctly, the problem is the specific script — find an updated version.

---

### ❌ Volcano Crashes on Launch (Windows)
**Cause:** Missing .NET 6.0 runtime or outdated Visual C++ Redistributable.
**Fix:** Install the latest Microsoft .NET Runtime (6.0 or higher) and the Visual C++ Redistributable 2022 x64, then relaunch Volcano as Administrator.

---

### ❌ Android APK Won't Install
**Cause:** "Install Unknown Apps" not enabled or not enough free storage on the device.
**Fix:** Settings → Security → enable Install Unknown Apps for your file manager. Ensure at least 200 MB of internal storage is free before installing the **volcano roblox apk**.

---

### ❌ iOS "Untrusted Developer" Error
**Cause:** The developer profile has not been trusted, or the sideloaded IPA has expired after 7 days.
**Fix:** Settings → General → VPN & Device Management → Volcano certificate → **Trust**. If expired, repeat the full sideload process with the current IPA from the Releases section.

---

### ❌ Aimbot Module Not Working
**Cause:** Aimbot requires the correct game character reference at inject time. If you injected before your character fully loaded, the targeting system has no valid anchor.
**Fix:** Reinject Volcano after your character has fully spawned. In the aimbot settings, confirm that the correct bone target (Head / HumanoidRootPart) is selected for the game you're playing.

---

### ❌ Auto-Update Not Triggering
**Cause:** Outbound connections to Volcano update servers are blocked by a firewall or VPN.
**Fix:** Add `VolcanoLoader.exe` to your firewall's outbound allow list. Temporarily disabling your VPN or switching to a mobile hotspot can confirm whether the issue is network-related.
